NAME
       sadbcat - Append/Create/Update encrypted SADB file.

SYNOPSIS
       sadbcat [-v] sadoor.db [ sadoor.db ... ] sash.db

DESCRIPTION
       sadbcat appends/creates/updates an encrypted database file
       (sash.db) from/with unencrypted sadoor.db files  generated
       with mksadb(8).

       This  database  is then used by sash(1) when communicating
       with the target machine running sadoor(8).

COMMAND LINE OPTIONS
       -v
           Verbose mode
